== Include a live console ==

Cypher with syntax highlighting and a live console:

[source,cypher]
----
START n=node(1)
RETURN n
----

.Console
[console]
----
(1) {"name":"A"}
(2) {"name":"B"}
(3) {"name":"C"}
(1)-[:KNOWS]->(2) {}
(1)-[:KNOWS]->(3) {}

start n=node(1) return n
----
